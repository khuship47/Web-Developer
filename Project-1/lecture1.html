Course Information

Description: This course is the first of a two-course sequence on Web Programming covering basic to advanced topics on Web Programming based on open standards and best practices. Topics covered include the HyperText Markup Language (HTML), specifying look and feel using Cascading Style Sheets (CSS), client-side programming in Javascript, server-side programming in PHP, and the Hypertext Transfer Protocol (HTTP).

Learning Outcomes

  • Build a simple website that organizes information effectively.
  • Identify an organization for information based on its inherent structure.
  • Use cascading style sheets to create style standards for a website.
  • Create a navigational framework that matches the content and genre of the site.
  • Explain why accessibility issues are an important consideration in web page development.
  • Explain separation of concerns as it applies to the design and implementation of a website.
  • Summarize the need and issues involved in website implementation and integration.
  • Explain and compare the inter-operability of formats.
