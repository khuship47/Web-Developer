All About Internet

Where is Internet?

  • Internet is basically a collection of internally connected networks.
  • It is common to visually represent internet as a cloud.
  • It is implemented via millions of copper wires and fiber optic wires connecting millions of computers and other networked devices.

Internet Connectivity

  • The broadband modem is a bridge between the network hardware outside the house and the network hardware inside the house. These devices are often supplied by the Internet Service Provider.
  • The wireless router is a hardware device that forward data packets from one network to another network.
  • Fiber junction boxes connect slower and older copper lines to higher bandwidth fiber optic cables.
  • Data centers requires high bandwidth due to which they are often directly connected to Internet Exchange Point.
  • Internet Backbone is the Tier 1 networks i.e., the first line of network. They use very fast fiver optic cable.
  • Regional networks have used less speed infrastructure.

Working in Web Development

Roles in Web Development

  • System Engineer: Develop and oversee all aspects of a complex system to solve a problem.
  • System Administrator: Make sures everything is up-to-date with software and hardware of the computer.
  • Data Architect: Plans how to implement or manage databases to make it more efficient.
  • Security Specialist: Implements security mesaure to protect computer networks and systems from attacks.
  • Developer: Helps create and test software from start to finish.
  • Front-End Developer: Builds the front-end portion of the websites and web applications i.e., the part users see and interact with.
  • Software Engineer: Use programming languages to build software solution for end users.
  • UX Designer: Makes the product more user friendly, enjoyable, and accessible.
  • Tester: Tests the product against various data set to see if it is ready to deploy or not.
  • Project Manger: Organizes, plans, and execute projects while working with restraints like budgets and schedules.
  • Business Analyst: Assess how companies are performing and help them improve their processess and systems.
  • Full-stack Developer: Develop full working website from back-end to front-end.
  • Nontechnical Roles: These includes customer services, marketing or sales, etc.
