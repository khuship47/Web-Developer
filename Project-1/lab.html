Hands-On Lab-1

Basics of HTML

  • Browser ignores extra spaces and paragrapha when adding in tags.
  • Tags like body, head, and html are not required in HTML, but is a good practice.
  • Any random tags added in HTML will be ignored by browser.
  • HTML5 is case insensitive.
  • Understanding of how to reference images from different folders using correct path.
  • Hyperlinks are an essential feature of any web page. Links are created via the anchor('a') element.
  • External link takes you to the website not in the current directory, while relative link takes to another page in the same website.
  • Learned about tags like 'a', 'br', 'div', 'img' as well as attributes like 'src', 'href', 'alt', 'title', etc.
